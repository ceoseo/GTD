## Features
- Tasks
- Projects
- Contexts
- Goals
- Status's
- Today/Overdue/Filtered View

## Tech Stack

- Next.js 13 App Directory
- Radix UI Primitives
- Tailwind CSS
- Icons from [Lucide](https://lucide.dev)
- Dark mode with `next-themes`
- Tailwind CSS class sorting, merging and linting.
- React Query
- Prisma
- Postgres
